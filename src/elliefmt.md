# EllieFMT

Formatter for the Ellie language

## Commands

- ### format
  `elliefmt format [file] -j`
    #### Options
    - `-j | --json-log`
    - Outputs the compilation log in JSON format
- ### analyze
  `elliefmt analyze [file] -j`
    #### Options
    - `-j | --json-log`
    - Outputs the compilation log in JSON format
