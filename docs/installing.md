# Installing Ellie

Ellie and all the ellie packages are installed by lia.

## For windows

Powershell

```powershell
blablalbal
```

## For Linux

Bash

```shell
blablalbal

```

## For Mac

Bash

```shell
blablalbal
```

After installing, you can run the following command to install ellie:

```shell
lia install ellie
```
