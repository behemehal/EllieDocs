# File Structure

```shell
-- lib
    | -- lib.ei
-- .lia
```

Let's explore the file structure of the ellie project. 
- `lib` is the folder where all the code belongs
- `lib/lib.ei` is the main file generated by default. 
- `.lia` is the configuration file for ellie.
